Para la ejecución individual se debe realizar de la forma:
Make compile
./PracticaConcurrente Ejemplos/EJEMPLO.dat <Threads> EJEMPLO.res

También se puede realizar
Make all
para realizar la compilación y la ejecución de todos los ejemplos con doce threads.
En el archivo MAKE FILE se encuentra la variable "Threads" y puede ser modificada en función de la cantidad de hilos que se deseen crear por ejecución
